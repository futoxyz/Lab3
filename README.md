# Лабораторная работа №3

## Запуск через uv
```
uv venv
uv sync
.venv/Scripts/activate
py -m src.main
```

# Ввод
Выбирается номер исполняемой операции

## Факториал и Фиббоначи
- При выборе поддерживается флаг `-r` для рекурсивного вычисления
- После выбора вводится число, над которым вычисляется выбранная функция
Для факториала счёт идёт от **0** элемента, для Фиббоначчи - от **1**
