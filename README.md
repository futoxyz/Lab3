# Структуры данных

## Stack
Реализован с помощью Node(value, next). Путь `/src/stack.py/`
- `push(x)` Добавить число `x` в конец
- `pop()` Достать число с конца
- `is_empty()` Проверка стека на наличие элементов
- `__len__` Длина стека
- `peek()` Посмотреть число с конца, не доставая его
- При использовании `pop()` или `peek()` для пустого стека - Ошибка

## Queue
Реализован с помощью Node(value, prev, next) - 2 ссылки у каждого элемента. Путь `/src/queue.py`
- `enqueue(x)` Добавить число `x` в конец
- `dequeue()` Достать число с начала
- `is_empty()` Проверка очереди на наличие элементов
- `__len__` Длина очереди
- `front()` Посмотреть число с начала, не доставая его
- При использовании `dequeue()` или `front()` для пустой очереди - Ошибка

# Обвесы

## Генерация тест-кейсов
Функции для генерации неупорядоченных массивов с числами. Путь `/src/arrays_generator.py`
- `rand_int_array(n, lo, hi, distinct, seed)` Генерация массива из `n` целых чисел от `lo` до `hi`. `distinct` означает отстуствие повторений (по умолчанию False). `seed` - Сид генерации
- `rand_float_array(n, lo, hi, seed)` Генерация массива из `n` чисел с плавающей точкой от `lo` до `hi`. `seed` - Сид генерации
- `nearly_sorted(n, swaps, seed)` Генерация массива из `n` целых чисел, где всего `swaps` смещений от отсортированного массива от `0` до `n-1`. `seed` - Сид генерации
- `many_duplicates(n, k_unique, seed)` Генерация массива из `n` целых чисел, где много повторений. Всего `k_unique` уникальных значений. `seed` - Сид генерации
- `reverse_sorted(n)` Генерация массива из `n` целых чисел от `n-1` до `0`
